## üìù The Student Assignment [cite: 190, 191]
**Assignment Goal:** To prove what helps and what harms a prompt, and to show improvement (or intentional deterioration) in performance using a graph[cite: 196].

### Work Stages [cite: 197]
1.  **Stage 1: Create a Dataset:** Create a dataset of question-answer pairs (e.g., sentiment analysis, multi-step math problems, logical statements)[cite: 198, 199, 200, 201, 202].
2.  **Stage 2: Measure Baseline:** Run the data with a basic prompt and measure statistical metrics (vector distances [cite: 206], histogram [cite: 207], mean and variance [cite: 208]).
3.  **Stage 3: Improve the Prompt:** Test the following methods[cite: 210]:
    * Improve a regular prompt (changes in system phrasing)[cite: 211].
    * **Few-Shot Learning** (adding up to 3 examples)[cite: 212, 213].
    * **Chain of Thought** ("Think step by step")[cite: 214].
    * **ReAct** (Optional): Integration with external tools[cite: 215].
4.  **Stage 4: Comparison and Presentation:** Present a **graph** showing the improvement (or deterioration) between the different prompt versions[cite: 219].

### Expectations [cite: 221]
* **Creativity** and original choice of problem[cite: 222].
* **Personal Insights** and explanations for why a change improved or harmed performance[cite: 223].
* **Critical Thinking** regarding the methodology[cite: 226].
* **Reminder:** The context is **Mass Production**, not a single use[cite: 232].
